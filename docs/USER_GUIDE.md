# TaskFlow – User Guide

Version: 1.0  
Author: Tovy REN and Jonathan Jalmain
Date: 18/02/26

---

## 1. Introduction

TaskFlow is a web-based project management application inspired by Trello.  
It allows users to manage projects using Kanban boards, organize tasks into lists and cards, assign members, track deadlines, and collaborate in real time.

This guide explains how to use the main features of the application.

---

## 2. Account Creation and Authentication

### 2.1 Register

1. Navigate to the registration page.
2. Enter:
   - Email address
   - Password
3. Click **Sign Up**.

Your account will be created and you will be redirected to your dashboard.

### 2.2 Login

Users can authenticate using:

- Email and password
- OAuth2 provider (e.g., Google)

After logging in, you will access your personal workspace.

---

## 3. Dashboard

The dashboard displays:

- Your boards
- Boards shared with you
- Options to create a new board

To open a board, simply click on it.

---

## 4. Creating a Board

1. Click on **"Create Board"**.
2. Enter a board name.
3. Confirm creation.

The board will appear in your dashboard and open automatically.

---

## 5. Managing Lists

Lists help organize tasks within a board (e.g., "To Do", "In Progress", "Done").

### 5.1 Add a List

1. Click **"Add List"**.
2. Enter the list title.
3. Confirm.

### 5.2 Edit or Delete a List

- Use the list options menu.
- Select edit or delete.

---

## 6. Managing Cards

Cards represent individual tasks within a list.

### 6.1 Create a Card

1. Click **"Add Card"** inside a list.
2. Enter a title.
3. Confirm.

### 6.2 Open Card Details

Click on a card to access detailed information.

Inside a card, you can:

- Add or edit a description
- Set a due date
- Assign members
- Add labels
- Post comments
- Change status

---

## 7. Drag & Drop Functionality

Cards can be moved:

- Between different lists
- Within the same list (to reorder)

Simply drag a card and drop it into the desired position.

This feature allows dynamic task management and workflow updates.

---

## 8. Member Management

Board administrators can:

- Invite members to a board
- Assign members to specific cards
- Manage user roles (Admin / Member)

Permissions vary depending on the assigned role.

---

## 9. Comments System

Each card includes a comment section.

Users can:

1. Write a comment.
2. Post it.
3. View comment history.

This allows collaboration and discussion directly on tasks.

---

## 10. Due Dates

A due date can be assigned to each card.

- Open the card
- Select a date
- Save

The due date helps track deadlines and project progress.

---

## 11. Real-Time Updates

EpiTrello supports real-time collaboration.

Changes such as:

- Card movement
- New comments
- Updates to tasks

are instantly visible to other connected users.

---

## 12. Gantt Chart

The Gantt view provides a timeline visualization of tasks.

It allows users to:

- View task durations
- Monitor deadlines
- Analyze project progression

The Gantt chart is generated automatically based on task dates.

---

## 13. Notifications

Users receive notifications for important events such as:

- Assignment to a card
- New comments
- Approaching deadlines

---

## 14. Logout

To log out:

1. Click on your profile icon.
2. Select **Logout**.

You will be redirected to the login page.

End of User Guide.
